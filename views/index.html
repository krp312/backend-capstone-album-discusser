Serving static assets with Express

This index.html page was sent over when we made a GET request to the root for this server.

In the head, we link to /style.css, which is stored on our server at public/style.css. Notice how we don't have to include `public` in the URL. Express just knows how to properly route this request, because we've enabled the public folder.

Below, we also link to /client.js, which is also located in the public folder of our server.
